# Changelog

## v1.4.0
Transferred to new account, fixed dependency vulnerabilities, improved admin tools, added private comment toggle on community posts, fixed reaction spam (transactional updates), replaced the single "Why I'm Doing This" reason with up to 10 separate reasons that appear as a swipeable carousel on Home under "Why I'm Fighting For Freedom", added an optional "Daily Scripture" setting (off by default) that shows one of 130 hand-picked Bible verses per day on the home screen, and added a "Themes" picker in Settings → Appearance with six aesthetic backdrops (Default, Beach, Mountain, Sunset, Forest, Night City) — each pairs a subtle full-screen background image with matching accent colors and syncs across devices.
